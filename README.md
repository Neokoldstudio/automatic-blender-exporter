# automatic-blender-exporter
A custom tool that automates heightmaps creation from blender
Simply add it as you would add any extensions to blender.

The tool will then sit in your tool bar, on the right of your viewport, in the hidden menu containing (Item | *Tool* | View | Animation)

To use it, simply select any objects you would like to bake in a height map, then click the "Bake Heightmap" Button.

you have the option to chose between the Evee and Cycles Renderer, wether to export in png or exr, with either 16 or 32 bit depth.
